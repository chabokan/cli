# Usage
<!-- usage -->
```sh-session
$ npm install -g @chabokan.net/cli
$ chabok COMMAND
running command...
$ chabok (-v|--version|version)
@chabokan.net/cli/0.1.0 darwin-x64 node-v12.16.1
$ chabok --help [COMMAND]
USAGE
  $ chabok COMMAND
...
```
